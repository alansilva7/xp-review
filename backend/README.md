# Game Review API

REST API para gerenciamento de jogos zerados e avaliados.

## Tecnologias
- Node.js
- Express
- TypeScript
- Prisma
- PostgreSQL

## Funcionalidades
- Criar jogos
- Atualizar jogos
- Deletar jogos
- Buscar por ID
- Listar jogos

## Rotas

POST /games
GET /games
GET /games/:id
PUT /games/:id
DELETE /games/:id
